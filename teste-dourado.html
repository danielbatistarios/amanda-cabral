Qual combinou melhor?

A (Azul) = o que existe hoje
B (Dourado) = dourado como accent principal
C (Híbrido) = dourado para elegância (tags, números, itálicos) + azul para ação (botões, CTAs)

O híbrido C separa funções: dourado = prestígio/autoridade, azul = ação/clique.
Assim o dourado não precisa "funcionar como botão" — só precisa brilhar onde importa.
